Frequently Asked Questions

Is my file data sent to any server?
No. All spreadsheet loading, JSON parsing, PDF generation, and CSV extraction are computed locally using javascript sandbox operations in your browser. We never upload your file.
Does this work offline?
Yes. Once cached, the Universal File Converter can be used without an active internet connection. Try disabling your internet and converting an Excel file!
