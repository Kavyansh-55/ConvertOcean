Percentages are everywhere: interest rates, business growth metrics, retail discounts, sales taxes, tips, and financial portfolios. However, calculating percentages, tracking percentage increases or decreases, and finding differences can be prone to manual calculation errors.

ConvertOcean's Percentage Calculator provides a suite of tools to calculate percentages, changes, and difference rates. It calculates all formulas locally in your browser memory, displaying step-by-step mathematical breakdowns with no data ever sent to a server.

How to Use the Percentage Calculator.

Our calculator provides four independent modules to solve different percentage problems:

  • Find a Percentage of a Number: Enter the percentage (X) and the base number (Y). The calculator outputs the exact portion (e.g. finding 15% of 200).
  • Find What Percentage X is of Y: Input a portion (X) and a total (Y). The tool calculates the percentage ratio (e.g. figuring out what percentage 30 is of 200).
  • Find Percentage Change (Increase/Decrease): Input an old value (X) and a new value (Y). The engine determines the percentage change and indicates if it is an increase or decrease.
  • Add or Subtract a Percentage: Input a percentage (X) and a base number (Y), then select Add or Subtract. The calculator outputs the final sum (useful for tips, markups, or discounts).

Percentage Formulas Explained.

The calculator displays the exact formulas used for each calculation mode:

  • Basic Percentage: Result = (Y × X) ÷ 100
  • Percentage Ratio: Percentage (%) = (X ÷ Y) × 100
  • Percentage Increase: Increase (%) = ((New Value − Old Value) ÷ Old Value) × 100
  • Percentage Decrease: Decrease (%) = ((Old Value − New Value) ÷ Old Value) × 100
  • Add Percentage: Result = Base + (Base × Percentage ÷ 100)

Since your file data remains inside sandboxed tab memory, it is never transmitted over the internet or uploaded to a remote server.
